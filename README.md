# AI-GESTURE-PRESENTATION-CONTROLLER
An AI-powered hand gesture–based presentation control system that enables touchless interaction using computer vision. Built with Python, OpenCV, and MediaPipe, the system supports real-time gesture recognition for slide navigation, virtual pointer control, drawing, and annotation without the need for physical input devices.
Implements a real-time AI-powered hand gesture recognition system
Enables touchless presentation control without physical input devices
Uses computer vision techniques for gesture detection
Detects hand landmarks using MediaPipe
Maps gestures to actions like:
Slide navigation
Virtual pointer movement
Drawing and annotation
Enhances user interaction and flexibility
Improves presenter mobility and audience engagement
Provides a cost-effective solution using only a standard webcam
Easy to use with minimal setup requirements
